## Poems

To create a new poem, create a file in the `poems/_posts` folder with the following format:

**File Name:**  
`YYYY-MM-DD-custom-name.md`

**File Content:**

```markdown
---
layout: post
title:  Poem 1
date:   2024-02-17 21:21:22 -0500
categories: poems
stars: 2
---

First Paragraph\
Second Line\
Third Line

Second Paragraph\
Fifth Line\
Last Line
```
