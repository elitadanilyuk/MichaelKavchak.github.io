Водиться кимось
Позбавлений основи
Українець
Без рідної мови.

Двері без замка,
Хата без дверей, 
Рот без язика —
Здобич легкА
Для підступних людей. 

Подвір’я без воріт
По кущах без чобіт,
Без ліхтарика вночі,
Без душі.
Бо слова чужі.
